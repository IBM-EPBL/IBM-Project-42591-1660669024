Handwritten Recognition System

Handwritten Text Recognition is a technology that is much needed in this worlds as of today. This digit Recognition systen is used to recognize the digits from different sources like emails, bank cheque, papers, images, etc. Before proper implementation we have relied on writing texts with our own hands which can results in errors. It is difficult to store and access physical data with efficiency. project presents recognizing the digits (0 to 9) from the famous MNIST dataset. Here will be using artificial neural networks/ convolution neural network.
